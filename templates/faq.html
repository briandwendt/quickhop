  What's a "3-letter IATA code?"

IATA is the International Air Transport Association, one of the world's largest airline industry trade associations. IATA helps formulate policy and standards.

An IATA airport code is a unique 3-letter designator for an airport. For example, the IATA code "BOS" designates Boston's Logan International Airport. You can search Wikipedia's List of Airports by IATA code if you need help remembering a code.
